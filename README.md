# 📚 Universal Paper Reader & Recommender Skill (通用论文研读与智能推荐助手)

一款专为高效泛读、攻克高难度/全英论文设计的 AI 智能体（Agent）配置方案。支持一键部署至 **Coze (扣子)** 平台，并无缝接入 **微信**、**飞书** 等日常办公软件。

无论面对的是量子力学、宏观经济，还是互联网产品运营论文，它都能通过**万能学术底层逻辑**，将长篇大论转化为一目了然的异色结构化骨架。

---

## ✨ 核心亮点

* 🎨 **多色万能拆解**：利用 Markdown 视觉隔离，用 🔴 红色标出核心痛点、🔵 蓝色拆解论证措施、🟢 绿色直击关键结论。
* 💡 **极简白话梗概**：拒绝学术黑话，150 字内将论文核心逻辑讲给外行听。
* 🎯 **目的导向评估**：自动指出论文的实际应用启发、局限性避坑指南、以及可直接跳过的冗长章节。
* 📚 **智能文献拓展**：自动推荐 【递进研究】、【经典必读】、【竞争/互补方案】三类高价值相关文献。

## 🛠️ 5分钟保姆级部署教程 (基于 Coze)

### 第一步：创建 Agent
1. 登录 [Coze 国内版 (coze.cn)](https://www.coze.cn)。
2. 进入“个人空间”，点击 **“创建 Agent”**。
3. 为它起一个名字（如：`论文全学科速读大师`），并简单填写简介。

### 第二步：配置灵魂提示词
1. 在 Agent 编排页面的正中间，找到 **“人设与回复逻辑” (Persona & Prompt)** 输入框。
2. 将本项目仓库中的 `instructions.md` 内的所有文本**完整复制并粘贴**进去。

### 第三步：装载谷歌学术插件
1. 在页面中的 **“插件 (Plugins)”** 栏目中，点击 **“添加插件”**。
2. 搜索 `Google Scholar` 并点击添加。

### 第四步：一键发布到 微信 / 飞书
1. 点击页面右上角的 **“发布 (Publish)”** 按钮。
2. 在发布渠道中，勾选你想要使用的平台（如 **飞书**、**微信客服** 等）。
